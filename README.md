### To start this app, run these commands below in two different terminal tabs.

## Start server
`npm run dev`

## Start webpack server
`npm run serve`

### Local DB stuff
1. install docker
2. run `docker-compose up -d` to spin up local db
3. run `docker-compose down` to destroy local db
